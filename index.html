Calculadora em Linguagem C

Projeto desenvolvido utilizando a linguagem C, onde foi implementada uma calculadora simples capaz de realizar operações básicas como adição, subtração, multiplicação e divisão.

Esse projeto foi importante para praticar conceitos de lógica de programação, estruturas condicionais, variáveis, entrada e saída de dados.
